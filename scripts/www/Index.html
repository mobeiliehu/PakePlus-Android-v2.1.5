设置
这是单文件版本（双击可开）。但是否能“直连调用接口”，取决于服务商是否允许浏览器跨域（CORS）。 若你用的是不支持 CORS 的接口，你需要改用支持 CORS 的服务或使用代理（例如 Vercel/Cloudflare Worker）。
预设
启用流式输出（SSE）
Chat API URL（完整到 /chat/completions）
API Key
文本模型（textModel）
视觉模型（visionModel，可选）
乔布斯式风格提示词（可改）
目标：短、硬、直，必要时带一点脏话强调；用事实与愿景“影响你”，而不是说教、命令或奉承。
乔布斯头像（本地保存）
默认是黑白乔布斯风格头像；你可替换。只存本地浏览器（IndexedDB），不上传。
主题卡片（可改 JSON）
格式：[{ "title": "...", "subtitle": "...", "starter": "...", "icon": "blue" }, ...]
说明：当你发送的消息包含图片时，本工具会优先使用 visionModel（若为空则退回 textModel）。 你的服务端必须支持 OpenAI 兼容的“多模态 message.content 数组”格式，否则可能报错。
